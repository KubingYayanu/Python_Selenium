# README

- 執行順序

```bash
$ pip3 install pipenv

$ mkdir project_folder
$ cd project_folder

# 依照 Pipfile.lock 來還原 Package
$ pipenv install

# 安裝 selenium
$ pipenv install selenium

# 進入虛擬環境
$ pipenv shell

# 執行 py
$ python xxx.py

# 離開虛擬環境
$ exit
```
